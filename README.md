# EngFlow
A frontend - only productivity web app designed for engineering students to manage academic tasks, exams and schedules with a clean and intuitive UI.
EngFlow is a frontend-only productivity web application designed specifically for engineering students.
It helps students manage academic schedules, track tasks, monitor exams, and visualize workload through a clean and intuitive dashboard.

Unlike generic productivity tools, EngFlow follows academic workflows such as subjects, labs, assignments, exams, and project reviews.
The application focuses on simplicity, usability, and distraction-free design to improve planning and time management for students.

Key Features:

- Academic calendar with dates and navigation
- Dashboard showing tasks, subjects, and exam overview
- Task tracking for assignments, labs, and projects
- Subjects and exams section with clear categorization
- Responsive and clean UI
- Frontend-only implementation

- Tech Stack Used

The project is built using basic frontend web technologies:

- HTML5 – Structure of the web pages
- CSS3 – Styling, layout, and responsive design
- JavaScript (Vanilla JS) – Page navigation, calendar logic, and interactivity

How to Run the Project

Option 1: Run Locally

1. Download or clone the repository
2. Open the project folder
3. Double-click on "index.html"
4. The app will open in your default web browser

Option 2: Run Using GitHub Pages

1. Go to Repository Settings
2. Click on Pages
3. Select:
   - Source: "Deploy from a branch"
   - Branch: "main"
   - Folder: "/root"
4. Save the settings
5. Open the generated GitHub Pages link to view the live project
